# EduScan Pro - Evaluador Inteligente de Exámenes

Sistema de evaluación automática de exámenes usando IA.

## 🚀 Características
- Evaluación automática con Gemini AI
- Interfaz moderna y responsive
- Exportación a Excel
- Compatible con modo oscuro
